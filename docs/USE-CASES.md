# Use Cases

## Phase 1 Productivity Utilities

These are the first workflows the app should support without admin rights.

### Document Work

- convert Word to PDF
- convert Excel workbooks to PDF
- convert PowerPoint decks to PDF
- convert images to PDF
- create zipped report bundles

### Reporting Work

- gather a reporting folder and export a shareable zip
- package monthly review decks
- archive approved files with a manifest

## Employer Branding Use Cases

### Campaign Prep

- convert approval decks to PDF before reviews
- create an approval pack from mixed source files and existing PDFs
- bundle campaign files for agency or stakeholder sharing
- package creative folders into clean zip handoffs
- prepare final review packs from mixed file formats

### Asset Management

- convert image sets into single-file PDFs for approvals
- archive old campaign folders with searchable manifests
- standardize export folders before publishing

### Content Operations

- bundle Meta campaign assets for internal review
- package supporting files for recruiter campaigns
- organize content review materials by campaign

### Meeting Support

- create one shareable pack for content review meetings
- convert latest slides to PDF before leadership review
- archive final approved material after sign-off

## Interaction Patterns

- drag a folder into the app to create a bundle
- drag convertible files into the app to convert them to PDF
- drag review assets into the app to create an approval pack

## Future Assistant Use Cases

These are for later phases and should stay opt-in.

### Contextual Help

- screenshot a problem and ask what to click next
- hold a hotkey to record a quick spoken instruction
- get spoken guidance back with visual points on screen

### Content Team Support

- point at a dashboard and ask for the next reporting step
- ask the assistant to summarize what is on screen
- get guided walkthroughs for repetitive publishing flows

### Safe Automation

- suggest the next click without auto-clicking
- highlight where the user should go in a complex tool
- help with training and onboarding workflows

## Enterprise Expansion

- outlook and calendar prep packs
- sharepoint and onedrive file packaging
- recurring report assembly
- transcript and subtitle preparation
- campaign archive automation
